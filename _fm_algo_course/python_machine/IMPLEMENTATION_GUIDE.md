# Python Kata Machine - Implementation Notes

## Instructions

This is a learning tool where you implement various data structures and algorithms. Each file in `src/day1/` contains stub methods that you need to implement.

### Testing Your Code

After implementing each data structure or algorithm, run its corresponding test to verify it works:

```bash
pytest tests/test_ArrayList.py -v
pytest tests/test_Stack.py -v
# ... and so on
```

### Tips

- Start with simpler data structures like Stack and Queue before moving to more complex ones
- Look at the test files to understand what each method should do
- Use Python type hints for better code clarity
- Consider edge cases like empty collections, None values, etc.

### Implementation Order (Suggested)

1. Stack - Simple and fundamental
2. Queue - Similar to Stack but FIFO
3. ArrayList - Core data structure
4. SinglyLinkedList - List implementation using nodes
5. DoublyLinkedList - Similar to singly, but with reverse links
6. Sorting algorithms - BubbleSort, QuickSort
7. Search algorithms - BinarySearch, LinearSearch
8. Tree algorithms - Binary tree traversals
9. Advanced structures - Trie, LRU, MinHeap

Good luck! 🚀
